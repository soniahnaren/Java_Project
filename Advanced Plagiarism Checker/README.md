# 🔍 Advanced Plagiarism Checker (Java)

## 📌 Project Overview

The **Advanced Plagiarism Checker** is a console-based Java application that compares two input texts and calculates their similarity using multiple algorithms.
It provides both **word-level and vector-based analysis**, along with **sentence matching**, making it a more accurate and insightful plagiarism detection tool.

---

## 🚀 Features

### 📊 Similarity Analysis

* Jaccard Similarity (word overlap)
* Cosine Similarity (vector-based comparison)

### 🔍 Text Processing

* Case-insensitive comparison
* Removal of special characters
* Word normalization

### 📄 Sentence Matching

* Detects identical sentences
* Displays matched content

### ⚡ Robust Handling

* Handles empty inputs safely
* Prevents divide-by-zero errors
* Clean and consistent output

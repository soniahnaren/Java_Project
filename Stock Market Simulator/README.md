# 📊 Stock Market Simulator (Java)

## 📌 Project Overview

The **Stock Market Simulator** is a console-based Java application that simulates real-world stock trading.
Users can buy and sell stocks, track their portfolio, and observe dynamic price changes over time.

This project demonstrates practical implementation of financial concepts using Java and provides an interactive trading experience.

---

## 🚀 Features

### 📈 Market Simulation

* Multiple predefined stocks
* Dynamic price updates (random fluctuations)
* Real-time stock price display

### 💰 Trading System

* Buy stocks based on available balance
* Sell owned stocks
* Quantity-based transactions

### 🧾 Portfolio Management

* Track owned shares
* View current stock values
* Display total portfolio worth

### 🏆 Financial Tracking

* Cash balance management
* Profit/Loss calculation
* Total investment value

---

## 🛠️ Technologies Used

* Java
* OOP Concepts (Classes, Objects)
* HashMap (for storing stocks and portfolio)
* Scanner (user input)
* Random (price simulation)

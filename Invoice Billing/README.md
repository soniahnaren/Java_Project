# 🧾 Advanced Billing System (Java)

## 📌 Project Overview

The **Advanced Billing System** is a console-based Java application designed to generate professional invoices with features such as item management, discount calculation, and GST computation.
It simulates a real-world shop billing system using a structured and menu-driven approach.

---

## 🚀 Features

* 🛒 Add multiple items with quantity and price
* 💸 Apply discount percentage
* 📊 Automatic subtotal calculation
* 🧾 GST calculation (18%)
* 📅 Invoice number and date generation
* 📄 Professional formatted bill output
* 🔁 Multiple billing sessions

---

## 🛠️ Technologies Used

* Java
* OOP Concepts (Classes, Objects, Methods)
* ArrayList (Java Collections Framework)
* Scanner (User Input)
* Date & Time API (`SimpleDateFormat`, `Date`)

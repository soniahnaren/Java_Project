# Special Number Checker-(Java)

This project is a menu-driven Java application that allows users to check whether a given number belongs to specific special number categories.

---

## Features

- Disarium Number Check  
- Fascinating Number Check  
- Keith Number Check  
- Menu-driven interface  
- Modular implementation using functions  

---

## Concepts Used

- Loops  
- Functions / Methods  
- Arrays  
- String manipulation  
- Mathematical logic  

---

## Menu Options
1. Disarium Number
2. Fascinating Number
3. Keith Number

---

## Definitions

### Disarium Number  
A number in which the sum of its digits powered with their respective positions is equal to the number itself.  
Example: 135 → 1¹ + 3² + 5³ = 135  

---

### Fascinating Number  
A number which, when multiplied by 2 and 3 and concatenated with the original number, results in a 9-digit number containing digits from 1 to 9 exactly once.  
Example: 192 → 192384576  

---

### Keith Number  
A number that appears in a sequence generated using its own digits, where each subsequent term is the sum of the previous digits.  
Example: 197 → 1, 9, 7 → ... → 197

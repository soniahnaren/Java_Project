# 📦 Warehouse Inventory System (Java)

## 📌 Project Overview

The **Warehouse Inventory System** is a console-based Java application designed to manage and track inventory in a warehouse.
It allows users to add, update, remove, and search items while also providing stock alerts and inventory valuation.

This project demonstrates real-world inventory management using efficient data structures and object-oriented programming.

---

## 🚀 Features

### 📦 Inventory Management

* Add new items
* Update item details (quantity, price)
* Remove items

### 🔍 Search Functionality

* Search items by name
* Display item details

### 📊 Stock Monitoring

* View complete inventory
* Low stock alerts for critical items

### 💰 Reporting

* Calculate total inventory value
* Track stock levels

---

## 🛠️ Technologies Used

* Java
* OOP Concepts (Classes, Objects, Encapsulation)
* HashMap (for efficient item storage and retrieval)
* Scanner (user input handling)

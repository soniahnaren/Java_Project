# 🎧 Advanced Music Playlist Manager (Java)

## 📌 Project Overview

The **Advanced Music Playlist Manager** is a console-based Java application that allows users to manage and interact with a music playlist efficiently.
It provides features similar to real music applications, including playback controls, playlist management, favorites, and history tracking.

---

## 🚀 Features

### 🎵 Playlist Management

* Add songs (title and artist)
* View complete playlist
* Remove songs

### 🔍 Search Functionality

* Search by song title
* Search by artist name

### ▶️ Playback Controls

* Play current song
* Next song
* Previous song

### 🔀 Play Modes

* Normal mode
* Shuffle mode
* Repeat mode

### ❤️ Favorites

* Mark songs as favorite
* View favorite songs

### 📜 History Tracking

* Track recently played songs
